# EclipseTrade

EclipseTrade est un projet personnel inspiré des plateformes de trading modernes.

## Aperçu

Le projet a été développé pour pratiquer :

- Next.js
- React
- Tailwind CSS
- Supabase
- Responsive Design
- UI/UX moderne

## Fonctionnalités

- Interface moderne et responsive
- Dashboard de trading fictif
- Authentification utilisateur
- Intégration Supabase
- Design dark mode
- Navigation fluide

## Technologies utilisées

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase

## Objectif du projet

Créer une expérience utilisateur moderne inspirée des plateformes financières professionnelles afin d'améliorer mes compétences en développement front-end et full-stack.
